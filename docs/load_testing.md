# Load testing

## 1. Конфиг

- **БД:** 1 000 пользователей у каждого 100 подписок => 100 000 подписок
- **Количество параллельных пользователей VU:** 32(2 * cpu_count)
- **Протокол:** gRPC / HTTP с valkey и без него
- **Сам теста:**
  - Ramp-up: 1 минута.
  - Stage: 5 минут.
  - Ramp-down: 30 секунд.
- **read/write oper:** 99/1
